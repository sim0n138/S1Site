# Руководство по развертыванию (Deployment Guide)

Это руководство поможет вам развернуть ваше Next.js приложение FitZen на продакшене. Мы сфокусируемся на **Vercel** как наиболее рекомендуемой платформе для Next.js, а также на **PostgreSQL** для базы данных.

## 1. Подготовка базы данных (PostgreSQL)

Для развертывания вам потребуется облачная база данных PostgreSQL. Рекомендуемые сервисы:

*   **Supabase:** Предоставляет бесплатный уровень и отлично интегрируется с Next.js/Prisma.
*   **Neon:** Современный, бессерверный PostgreSQL с хорошей интеграцией.
*   **ElephantSQL / Render:** Классические варианты хостинга PostgreSQL.

### Шаги:

1.  **Создайте базу данных** на выбранном вами сервисе.
2.  **Получите строку подключения (Connection String)**. Она будет выглядеть примерно так:
    \`\`\`
    postgresql://[USER]:[PASSWORD]@[HOST]:[PORT]/[DATABASE]?schema=public
    \`\`\`
3.  **Выполните миграцию Prisma:** После получения строки подключения, вам нужно выполнить миграцию, чтобы создать все таблицы (`User`, `Workout`, `UserProfile` и т.д.) в вашей облачной БД.

    Установите строку подключения как переменную окружения и выполните:
    \`\`\`bash
    # Замените <YOUR_DATABASE_URL> на вашу строку подключения
    export DATABASE_URL="<YOUR_DATABASE_URL>"
    npx prisma migrate deploy
    \`\`\`

## 2. Развертывание на Vercel

Vercel — это платформа, созданная для Next.js, и она обеспечивает максимально простое развертывание.

### Шаги:

1.  **Загрузите изменения:** Убедитесь, что все последние изменения (включая CRUD-функционал, который мы добавили) закоммичены и отправлены в ваш репозиторий GitHub.
    *   *Это уже сделано в коммите \`feat: Implement basic User CRUD API and frontend component\`.*
2.  **Импортируйте проект:**
    *   Перейдите на [Vercel Dashboard](https://vercel.com/dashboard).
    *   Нажмите **"Add New"** -> **"Project"**.
    *   Выберите ваш репозиторий `sim0n138/S1` из списка.
3.  **Настройте переменные окружения:**
    *   На экране настроек проекта Vercel перейдите в раздел **"Environment Variables"**.
    *   Добавьте переменную с ключом `DATABASE_URL` и значением — вашей строкой подключения к облачной PostgreSQL (из Шага 1).
4.  **Развертывание:**
    *   Нажмите **"Deploy"**. Vercel автоматически определит, что это Next.js проект, установит зависимости, соберет проект и развернет его.

## 3. Проверка функционала

После успешного развертывания:

1.  Перейдите по URL вашего нового сайта.
2.  Проверьте, что форма создания пользователя работает, и данные сохраняются в вашей облачной базе данных.
3.  Проверьте, что пользователи отображаются в списке и их можно удалить.
